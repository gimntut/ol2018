В начало

Сколько отрицательных

Ограничение времени 1 секунда
Ограничение памяти 64Mb
Ввод negative.in
Вывод negative.out

4. Задана матрица чисел размером n на m. Вам нужно найти номер последнего столбца массива, в котором находится наибольшее количество отрицательных элементов.

Формат ввода

В первой строке входного файла negative.in записаны через пробел два числа n и m, соответственно количество строек и столбцов матрицы (2<n,m<501). В последующих n строках записаны через пробел m чисел – элементы матрицы.(-1000<a[i,j]<=1000)

Формат вывода

В выходной файл вывести negative.out одно число – номер столбца в которой наибольшее количество отрицательных элементов. Если таких стольцов несколько выведите с наибольшим номером.

Пример

Ввод Вывод 
3 4
-1 -9 0 -7 
0 -8 3 -5 
7 5 3 -10 

4
